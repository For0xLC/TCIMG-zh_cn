Дополнение №9

Дополнение к командам creat, marks и многим другим.


Получение части пути:

ЗначениеОписание
0,1полный путь
2,3расширение
4,5имя без расширения
6,7имя с расширением
8,9 путь без расширения
10,11 путь родительской папки с конечным слэшем
12,13 путь родительской папки без конечного слэша
14,15 путь дедушкиной папки с конечным слэшем
16,17 путь дедушкиной папки без конечного слэша
18,19 диск с конечным слэшем
20,21 диск без конечного слэша
22,23 путь без диска с начальным слэшем
24,25 путь без диска без начального слэша
26,27 путь без диска и корня с начальным слэшем
28,29 путь без диска и корня без начального слэша
30,31 расширение с точкой впереди
32,33 имя с расширением для файлов, для папок имена с конечным слешем
34,35 расширение, если папка, то возвращается "\"
36,37 имя без расширения для файлов, для папок имена с конечным слешем
38,39 пустая строка для файлов, имена для папок
Замечание:
0-39 - нечётный номер заключает часть пути в кавычки.
