Дополнение №4

Дополнение к командам menug, menux, menuc, mxmnz, tityn, wmenu, autrn, title, prior, zmenu и другим


Autorun - это простой контентный плагин для выполнения некоторых действий при запуске Total Commander. Можно запустить файл, выполнить команду (cm_ и em_), установить переменную окружения, записать значение в ini-файл, задать задержку между выполнением команд, загрузить библиотеку или плагин. Сейчас в основном разрабатывается для удобства изготовления портабельной сборки. Он покрывает практически весь необходимый функционал автозапуска.
Autorun представляет собой простой командный процессор, выполняющий команды, заданные в файле конфигурации autorun.cfg . Все команды выполняются в отдельном потоке и не должны завешивать Total Commander даже в случае проблем с одной из ней.

Автор: Loopback

Примеры запуска вместе с Total Commander некоторых команд TCIMG, путём внесения в конфигурационный файл autorun.cfg необходимых строк:

# запуск информационной строки и прикрепление её к Total Commander, настройки берутся из TCIMG.ini из секции [TitleWindow3]
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "infow=0||TitleWindow3||attach<@%AUTORUN_TCHANDLE%>"

# запуск информационной строки и прикрепление её к Total Commander
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "infow=0||attach<@%AUTORUN_TCHANDLE%>"

# запуск панели и прикрепление её к Total Commander
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "xbarx=%COMMANDER_PATH%\Files\Bar\Example_1.bar||attach<TTOTAL_CMD|0|0>"

# запуск настроенного меню для трея вместе с Total Commander
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "zmenu=tray_Menu_Example"

# запуск настроенного меню (вызов по горячим клавишам) вместе с Total Commander
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "zmenu=hotkey_Menu_Example"

# установить приоритет "выше среднего" Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "prior=3||$%AUTORUN_TCPID%"

# установить приоритет "высокий" Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "prior=4||$%AUTORUN_TCPID%"

# установить приоритет "ниже среднего" Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "prior=1||$%AUTORUN_TCPID%"

# скрыть меню Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "wmenu=2"

# Выключение панели инструментов при запуске Total Commander
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "elmof=1"

# Выключение кнопок дисков при запуске Total Commander
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "elmof=2"

# Выключение командной строки при запуске Total Commander
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "elmof=11"

# скрыть заголовок Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "tityn=0"

# скрыть меню и заголовок Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "tityn=0 wmenu=2"

# скрыть меню, заголовок и развернуть Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "ssize=1 tityn=0 wmenu=2 mxmnz=0"

# установить в меню расцветку 1 Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "menuc=1"

# установить заданным пунктам меню состояние недоступности (отображение серым цветом) Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "menug=0||1,3,5||2,6||3,2,1"

# установить состояние недоступности пунктам меню Total Commander с командой em_Grey при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "menux=1||em_Grey"

# установить длину горизонтальной полосы 700px в файловых панелях Total Commander, если включён краткий режим или режим комментариев
Sleep 500
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "horiz=3||700 horiz=4||700"

# открыть Total Commander развёрнутым во весь экран поверх таскбара при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "ssize=1 wpull=0,0,@W,@H ontop=1"

# установить название заголовка "Привет" Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "title=Привет"

# установить название заголовка "Total Commander 8.5 My_Sborka 888" Total Commander при запуске
ShellExec "%COMMANDER_PATH%\TCIMG.exe" "title=Total~~Commander~~8.5~~My_Sborka~~888"
