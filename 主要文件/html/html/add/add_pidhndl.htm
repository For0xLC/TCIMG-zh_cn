Дополнение №36

Дополнение к командам и параметрам, связанных с окнами и процессами.


Универсальное получение PID (идентификатор) процесса или Handle (дескриптор) окна.

В тех командах и параметрах, где указано Дополнение №36 PID и Handle можно получить относительно любых предложенных вариантов:

  • Получение ИД процесса TCIMG. PID, пример:
      @AutoItPID

  • Получение через ИД процесса. PID, пример:
      - 6328

  • Получение через Дескриптор окна. Handle, примеры:
      - 0x000102F8
      - 66296

  • Получение через Имя процесса. NameProcess, примеры:
      - Totalcmd.exe
      - AkelPad.exe
      - notepad.exe

  • Получение через Путь процесса. PathProcess, примеры:
      - E:\TC IMAGE\Totalcmd.exe
      - E:\TC IMAGE\AkelPad.exe

  • Получение активного окна, примеры:
      - ACTIVE
      - [ACTIVE]

  • Получение через Класс окна. NameClass, примеры:
      - TTOTAL_CMD
      - AkelPad4

  • Получение через Класс окна. [CLASS:NameClass], примеры:
      - [CLASS:TTOTAL_CMD]
      - [CLASS:AkelPad4]

  • Получение через Класс окна {NameClass}, примеры:
    (для программ с множеством дочерних процессов и одинаковых классов позволяет получить верхнее отображаемое/скрытое окно), примеры:
      - {MozillaWindowClass}
      - {Chrome_WidgetWin_1}

  • Получение через Заголовок окна [TITLE:TextTitle], примеры:
      - [TITLE:Справка~~TCIMG]
      - [TITLE:Total~~Commander]

  • Получение через Класс и Заголовок окна [CLASS:NameClass;TITLE:TextTitle], примеры:
      - [CLASS:HH~~Parent;TITLE:Справка~~TCIMG]

  • Получение через Класс и Заголовок окна NameClass**TextTitle, примеры:
      - HH~~Parent**Справка~~TCIMG

  • Получение через Класс окна, с помощью регулярных выражений [REGEXPCLASS:RegExp], примеры:
      - [REGEXPCLASS:Afx:400000:b:10003:6:.*]
      - [REGEXPCLASS:.*Window.*]

  • Получение через Заголовок окна, с помощью регулярных выражений [REGEXPTITLE:RegExp], примеры:
      - [REGEXPTITLE:.*Блокнот]
      - [REGEXPTITLE:.*HTML.*]
