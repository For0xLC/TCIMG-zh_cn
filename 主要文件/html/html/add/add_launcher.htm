Дополнение №27

Лаунчер TCIMGGO.exe - запуск утилиты с заданными параметрами.


С версии 10.3 в архив с утилитой входит файл TCIMGGO.exe, который позволяет запускать TCIMG.exe с заданными параметрами.

Для того, чтобы настроить запуск с нужными параметрами необходимо:
1. Дублировать файл TCIMGGO.exe
2. Переименовать базовое имя файла дубликата (к примеру, в TIMESX.exe).
3. В TCIMG.ini (TCIMGU.ini) в секции [Commands] прописать именной ключ. Именем ключа является базовое имя (без расширения) переименованного лаунчера (в нашем случае TIMESX)
4. В ключе прописать необходимые параметры вызова.

Пример записи:

[Commands]
TIMESX=timesx
COLOR=color=fullcolor

Теперь при запуске TIMESX.exe будет запускаться "Таймер. Секундомер. Будильник." (команда timesx)

Если переименовать дубликат TCIMGGO.exe в COLOR.exe и прописать параметры, то будет запускаться приложение "Просмотр и выбор цвета"

Дальнейшая настройка вызова зависит от задачи и фантазии.


Добавлено в версии 26.2

По умолчанию TCIMGGO.exe запускает утилиту TCIMG.exe с заданными параметрами и завершает свой процесс.
Для того чтобы TCIMGGO.exe оставался в системе и ожидал завершения запущенного TCIMG.exe необходимо вначале строки параметров прописать символ "@"

Пример записи:

[Commands]
TIMESX=@timesx
COLOR=@color=fullcolor



TCIMGGO.exe
  • Может быть сколько угодно дубликатов файла.
  • Файлы могут находится как в одной папке с TCIMG.exe, так и в подпапках утилиты на любом уровне.
