Файл конфигурации TCIMG.ini

В TCIMG.ini можно настроить дополнительные опции (ключи) для различных команд.

Секция [Setting]

Основные настройки.

LanguageЯзык отображаемый в диалогах: значение является секцией в языковых файлах .lng (Language=Russian русский по умолчанию) подробности.
LNGПодключение пользовательских файлов lng с командами утилиты. Прописывается либо полный путь, либо имя файла, который находится в папке ...\Files\Lng\, подробности. Файлы разделяются /
TCGПодключение пользовательских файлов tcg с командами утилиты. Прописывается либо полный путь, либо имя файла, который находится в папке с утилитой. Файлы разделяются /
MsBoxOrCounterПри совпадении имён команд копирования/переименования/перемещения...
  0 - включается счётчик, который добавляется к базовому имени.
  1 - вызывается диалоговое окно для ввода нового имени.
CounterRankМинимальное количество цифр в счётчике.
CounterDelimeterРазделитель между именем и счётчиком.
CounterLeftСимвол/строка, добавляемая слева от счётчика.
CounterRightСимвол/строка, добавляемая справа от счётчика.
ReplaceClipboardСтрока предварительной замены недопустимых символов в именах объектов в случае, когда имя берётся из буфера обмена (при переименовании из тегов html); где пара: недопустимый символ и чем его заменять разделяются точкой; между парами необходимо ставить две точки. Если необходимо удалять символ, то вместо замены прописывается тег @Z

Режим при масштабировании системы:
  0 - ничего не менять (при этом в ряде команд и параметрах, связанных с координатами, могут возвращаться неправильные значения).
