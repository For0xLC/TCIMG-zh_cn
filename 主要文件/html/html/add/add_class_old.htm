Дополнение №17+

Список классов окон и элементов Total Commander 8.01 32 и 64 разрядности.

Информация взята из файла ...\Files\Ini\TCIMG3264_8_01.ini

Для пользователей версией Total Commander 8.01, утилита автоматически определит версию и соответствующий файл

Утилита автоматически определяет разрядность запущенного Total Commander.
Утилита автоматически перебирает классы для получение информации (где это необходимо).

Для внесения в команду следует:
1. Найти нужный класс, контролл.
2. Вставить в параметр значение ключа, приписав в начале +tc

Пример записи:

+tcTOTAL1


Отдельное спасибо Asmadeus за составление классов для 64 разрядности.
