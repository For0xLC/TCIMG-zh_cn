2. Список идентификаторов известных GUID папок.
   (Получение пути зависит от системы и установленных компонентов)

Для получения всех абсолютных путей GUID можно воспользоваться командой infos:

infos=2||23 ;; отображение информации системных GUID папок

Для получения заданного абсолютного пути GUID можно воспользоваться командой infos:

sends=@<info=s260|39> GLOBALSENDS<a> ;; получить путь папки Music (пользовательский путь)

sends=@<info=s260|39|1> GLOBALSENDS<a> ;; получить путь папки Music (путь по умолчанию системы)

sends=@<info=s260|{4BD8D571-6D19-48D3-BE97-422220080E43}> GLOBALSENDS<a> ;; получить путь папки Music (пользовательский путь)

sends=@<info=s260|{4BD8D571-6D19-48D3-BE97-422220080E43}|1> GLOBALSENDS<a> ;; получить путь папки Music (путь по умолчанию системы)
