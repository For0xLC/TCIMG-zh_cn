Дополнение №6

Дополнение к командам infos=3||1||File и <info=o250File|Numbers>.


Получение информации папки или файла с помощью метода GetDetailsOf (ShellDetails).

  !!! Может отличаться относительно версий системы Windows !!!

Для получения названий типов информации и номеров воспользуйтесь командами:

; отображение всей не пустой информации GetDetailsOf объекта с названиями и номерами типа информации
infos=3||20||File
; отображение всей информации GetDetailsOf объекта с названиями и номерами типа информации
infos=3||21||File
